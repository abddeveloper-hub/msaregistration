Import May 2026 Attendance

This script will read the transcription below and insert attendance records for 1st Dars and 2nd Dars into Firestore.

Format: RollNo: mark,mark,mark... (19 marks for days: 4,5,6,7, 9,10,11,12,13, 15,16,17,18,19, 21,22,23,24,25)

Marks: x = present, A = absent, - = skip, anything else = leave.
